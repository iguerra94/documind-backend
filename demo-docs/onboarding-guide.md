# Developer Onboarding Guide

## Purpose

This guide helps new developers understand the platform and get started locally.

## Main services

The platform is composed of:

- orders service
- payments service
- notifications service

## Local setup

Before starting locally, developers must:

1. Configure environment variables
2. Start required infrastructure dependencies
3. Verify authentication settings
4. Check service-to-service connectivity

## Recommended learning order

New developers should first read:

1. Authentication Guide
2. Orders Service
3. Event Flow
4. Payments Runbook
